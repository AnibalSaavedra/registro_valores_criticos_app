# Registro de Notificación de Valores Críticos

Esta aplicación permite registrar notificaciones de valores críticos en un laboratorio clínico, manteniendo una base de datos persistente en Excel.

## Valores Críticos Considerados:
- INR ≥ 3.5%
- Hematocrito ≤ 16%
- Glucosa ≤ 40 mg/dL
- Potasio ≥ 6.2 mEq/L
- Troponina ≥ 0.12 ng/mL

Cada registro incluye datos del paciente, del examen crítico, contacto notificado, tiempo de respuesta y estado del reporte.

La app fue desarrollada por Anibal Saavedra para uso interno del laboratorio de Clínica Río Blanco.
